# public-scripts
Repo for public PlasHost scripts.

## Getting the Scripts
### `helpful` script.
**Installation**<br></br>
Run the following command:
```bash
curl -s "https://raw.githubusercontent.com/PlasHost/public-scripts/main/helpful.sh" | bash -s -- i
```
**Updating**<br></br>
Run the following command:
```bash
curl -s "https://raw.githubusercontent.com/PlasHost/public-scripts/main/helpful.sh" | bash -s -- u
```
### `db_copy.sh` script.
Run the following command:
```bash
curl -o "https://raw.githubusercontent.com/PlasHost/public-scripts/refs/heads/main/db_copy.sh" "db_copy.sh"
```
